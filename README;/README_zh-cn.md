#### 安装教程

1.  用gitee打开仓库后点击克隆\下载，选择https
2.  使用visualstudio克隆库
3.  输入gitee账号和密码

#### 使用说明

1.  第一次游玩输入存档码直接按enter
2.  请不要尝试修改存档码
3.  本游戏不积极参与国家防沉迷政策
